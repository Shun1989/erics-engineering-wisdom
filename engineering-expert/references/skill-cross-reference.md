## 跨域协作协议

当用户的问题可能跨域时，按以下流程处理：

```
1. 识别问题关键词 → 匹配可能的技能组合（查本表）
2. 按优先级加载对应技能的 references 文件
3. 综合各领域知识给出系统性回答
4. 在回答中明确标注"以下内容综合了 XX 和 XX 两个领域的知识"
5. 如问题明确偏向某一领域，建议用户"这个问题更深入的部分可以加载 XX 技能继续分析"
```

### 加载示例（motor-expert + nvh-expert 联动）

用户问："电机在高速时出现啸叫声，怎么排查？"
→ motor-expert 计算特征频率（f_s = n/60, f_c, f_harmonic）
→ nvh-expert 提供3D图谱分析、阶次分析、BI评级
→ 联合诊断：判断是电磁音/齿轮音/结构共振

## 冲突处理

当两个技能对同一问题给出不同侧重时：
1. 优先采纳与用户问题具体描述更匹配的技能视角
2. 如果用户涉及某个物理系统（如"齿轮箱"），优先加载 gear-expert的详细参考
3. 在回答中说明"从XX角度看是……从XX角度看是……综合来看……"
