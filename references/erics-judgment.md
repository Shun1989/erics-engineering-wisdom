## 第一部分：项目案例库（Case Studies）

### 案例1：BMW IBK2 高频音（5379Hz刺耳音）

| 字段 | 内容 |
|:-----|:------|
| **项目** | BMW IBK2 |
| **问题** | 5379Hz刺耳高频音 |
| **诊断过程** | 频谱分析定位异常频率→计算电机特征频率→比对发现机壳固有频率约5300Hz，与5379Hz差值仅79Hz→判断为结构共振 |
| **根因** | 机壳5300Hz模态与电机5379Hz特征频率共振 |
| **解决方案** | 碳刷宽度从3.5mm优化到2.8mm→电流波动从1.35A降到0.37A |
| **效果** | 噪声降低72.6% |
| **教科书上没有的** | "高频刺耳音的根因往往不在齿轮，而在电机电磁参数与结构的耦合。碳刷宽度改变电流波动比做隔振更有效" |
| **可复用原则** | 电机高频噪声查三条：碳刷参数→电磁力波→机壳模态 |

### 案例2：BMW IBK2 蛐蛐声

| 字段 | 内容 |
|:-----|:------|
| **项目** | BMW IBK2 |
| **问题** | 大批量蛐蛐声 |
| **诊断过程** | 频谱分析→3517Hz异常→溯源至蜗杆刀纹频率 |
| **根因** | 蜗杆加工有45个刀纹→刀纹频率 = 蜗杆转频×45 = 3517Hz |
| **解决方案** | 改进蜗杆刀纹工艺 |
| **效果** | 彻底解决 |
| **教科书上没有的** | "蜗杆刀纹引起的噪声最容易误诊为齿轮啮合问题。关键区分：蜗杆刀纹频率 = 转频 × 刀纹数，而啮合频率 = 转频 × 齿数——差一个齿数和刀纹数的关系" |
| **可复用原则** | 蜗杆噪声先与刀纹数和齿数对比排除 |

### 案例3：捷豹路虎 XVS3 循环音

| 字段 | 内容 |
|:-----|:------|
| **项目** | 捷豹路虎XVS3 |
| **问题** | 循环音（0~10Hz周期性音调波动） |
| **诊断过程** | 3D图谱→发现马达特征频率被蜗轮频率调制 |
| **根因** | 马达特征频率与蜗轮频率耦合产生调制 |
| **解决方案** | 降低骨架内阻 + 提高马达性能 |
| **效果** | Modulation ≤5%，客户可接受 |
| **教科书上没有的** | "循环音的根本不是齿轮做得不好，而是电机-齿轮系统的电气-机械耦合。先优化电气参数（内阻），再改机械方案" |
| **可复用原则** | 循环音先查电气参数再查机械，成本路径正好反过来 |

### 案例4：Mazda J72Y Buzzing噪音

| 字段 | 内容 |
|:-----|:------|
| **项目** | Mazda J72Y |
| **问题** | 低频Buzzing噪音（800Hz以内） |
| **诊断过程** | 3D图谱看到条状光亮带→跟踪到软轴侧 |
| **根因** | 软轴装配过松→波浪形传动 |
| **解决方案** | 优化软轴装配 |
| **效果** | 解决 |
| **教科书上没有的** | "Buzzing（低频嗡嗡声）在座椅电机中80%是软轴问题，不是电机或齿轮问题。软轴的排查成本最低——目视检查即可，不要上来就拆电机" |
| **可复用原则** | 低频Buzzing：先看软轴装配，再看齿轮啮合，最后看电机 |

### 案例5：通用GEM 响度超标

| 字段 | 内容 |
|:-----|:------|
| **项目** | 通用GEM |
| **问题** | 响度3.332sone，但目标≤2.5sone |
| **诊断过程** | FFT→锁定100~300Hz贡献频段→拆解到软轴+齿轮箱两个源头 |
| **根因** | 软轴装配过松 + 齿轮箱啮合不良（双重根源） |
| **解决方案** | 优化软轴装配 + 优化齿轮啮合 |
| **效果** | 响度达标（实测≤2.5sone） |
| **教科书上没有的** | "响度超标时不要只找一个根因——很多时候是两个源叠加。而且先处理软轴（低成本），如果还不够再动齿轮（高成本）。这个优先级在哪里都适用" |
| **可复用原则** | 响度超标拆解到多个源，按成本排序处理 |

### 案例6：Mazda GEN7 ED 敲击异音

| 字段 | 内容 |
|:-----|:------|
| **项目** | Mazda GEN7 ED |
| **问题** | 9.13Hz规律敲击异音 |
| **诊断过程** | 计算参数→发现9.13Hz对应行星架转频→检查盖板与行星架配合 |
| **根因** | 盖板孔径过大→行星架晃动→周期性敲击 |
| **解决方案** | 优化盖板孔径 |
| **效果** | 消除 |
| **教科书上没有的** | "9.13Hz这个频率太低了，大多数人会忽视。但行星齿轮的敲击频率 = 行星架转频，就是那么低的。低频敲击先算行星齿轮的转频" |
| **可复用原则** | 行星齿轮低频敲击→检查行星架与盖板配合间隙 |

### 案例7：BMW IBK2 Tilt2000 苍蝇叫异音

| 字段 | 内容 |
|:-----|:------|
| **项目** | BMW IBK2 Tilt2000 |
| **问题** | 223Hz苍蝇叫异音 |
| **诊断过程** | 频率计算→对齿数表→锁定到衬套侧（61齿） |
| **根因** | 衬套打滑 |
| **解决方案** | 优化衬套尺寸 |
| **效果** | 消除 |
| **教科书上没有的** | "223Hz这个频率正好落在人耳最敏感的范围（200-500Hz），所以客户很容易感知。对这类频率的排查优先级应该高于高频" |
| **可复用原则** | 人耳敏感频段（200-500Hz）的问题哪怕幅度不高也要优先处理 |

### 案例8：BMW IBK2 高调晃动

| 字段 | 内容 |
|:-----|:------|
| **项目** | BMW IBK2 |
| **问题** | 3.5Hz高调晃动 |
| **诊断过程** | 追踪到齿轮箱运转频率 |
| **根因** | 齿轮箱运转频率与结构耦合 |
| **解决方案** | 对应零件优化 |
| **效果** | 解决 |
| **教科书上没有的** | "3.5Hz属于人体最敏感的振动频率（腹部共振4-8Hz，脊柱10-12Hz）。这种低频晃动客户不会说'好吵'，但会说'好不舒服'——容易被误诊为机械问题而不是NVH问题" |
| **可复用原则** | 低频振动（<10Hz）优先查人体振动标准ISO 2631，不是查噪声 |

### 案例9：MQB项目 倍频程超标

| 字段 | 内容 |
|:-----|:------|
| **项目** | MQB |
| **问题** | 1/3倍频程在524.1Hz处超标 |
| **诊断过程** | 频谱→524.1Hz→与电机特征频率525Hz比对→仅差0.9Hz |
| **根因** | 电机特征频率525Hz激励支架共振 |
| **解决方案** | 电机与支架间加橡胶隔振 |
| **效果** | SPL降低3-4dB，倍频程达标 |
| **教科书上没有的** | "电机特征频率与结构频率差值<1Hz就是共振，但很多人只知道看模态匹配。实际经验：加橡胶隔振是最快最便宜的方案，不用改模具" |
| **可复用原则** | 电机共振优先试试隔振垫（橡胶/硅胶），成本低见效快 |

## 第二部分：工艺窍门（Process Tips）

| 领域 | 条件 | 窍门 | 理由 |
|:-----|:-----|:-----|:-----|
| NVH诊断 | 任何电机噪声 | 先空载听→再加载听→比较差异 | 空载消失=齿轮/负载侧问题；空载还在=电机本体问题 |
| 座椅传动 | 低频Buzzing | 先目视检查软轴装配是否过松 | 80%的Buzzing是软轴问题，目视10秒搞定 |
| 电机 | 有刷电机哒哒声 | 先拆碳刷，不看长度就看端面磨损状态 | 端面倾斜磨损比长度更说明问题 |
| 齿轮 | 齿轮噪声初诊 | 用手机录一段10秒音频，数每秒"哒"几下 | 频率=节奏×60，不用仪器就能估算啮合频率 |
| 声学测量 | 3D图谱分析 | Ctrl+Shift+↑↓调到5-20Hz带宽 | 太宽看不清，太窄看不到 |
| 声品质 | 响度超标 | 先锁定额外的贡献频带再动手 | 盲目优化效率低，先锁定再动手 |
| 振动 | 人体感知振动 | 优先做速度一次积分，不要只看加速度 | 人体对速度变化比对加速度更敏感 |

## 第三部分：决策准则（Decision Rules）

```
IF 电机异响 AND 空载出现 THEN 优先查碳刷/电刷

IF 电机异响 AND 加载出现 THEN 优先查齿轮/负载

IF 低频Buzzing(<800Hz) THEN 先查软轴装配

IF 循环音(0-10Hz波动) THEN 先优化电气参数再改机械

IF 高频刺耳音(>3000Hz) THEN 查碳刷参数+电磁力波+机壳模态

IF 行星齿轮敲击 THEN 查行星架与盖板配合间隙

IF 电机共振 THEN 先试橡胶隔振垫

IF VDI评级<7 THEN 必须处理

IF VDI评级≥7 THEN 与客户Workshop确认是否接受
```
