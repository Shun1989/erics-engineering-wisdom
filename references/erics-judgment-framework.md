## 一、个人经验的结构化框架

### 类型A：项目案例（Case Study）

| 字段 | 说明 | 示例 |
|:-----|:-----|:-----|
| **项目名称** | 客户+产品代号 | 通用GEM |
| **时间** | 项目起止 | 2023 Q3-Q4 |
| **问题描述** | 背景+症状+影响 | 座椅调节器响度超标，目标≤2.5sone，实测3.33sone |
| **诊断过程** | 怎么查的，用了什么方法 | FFT分析→锁定100~300Hz贡献频带→拆解到软轴和齿轮箱 |
| **根因** | 最终找到的根本原因 | 软轴装配过松+齿轮啮合不良 |
| **解决方案** | 做了什么 | 优化软轴装配工艺 + 改善啮合 |
| **效果** | 量化结果 | 响度达标，BI评级达标 |
| **教科书上没有的** | ⭐核心价值——你的独特判断 | "响度超标时大多数人会先调齿轮修形，但我们发现软轴装配的影响更大，而且成本更低" |
| **可复用的原则** | 这类问题下次可以怎么套用 | "座椅传动件异响：先查装配（低成本高概率），再查齿轮/电机（高成本）" |

### 类型B：工艺窍门（Process Tips）

> 一句话说清楚：什么条件下 + 怎么做 + 为什么

| 领域 | 条件 | 窍门 | 理由 |
|:-----|:-----|:-----|:-----|
| NVH | 座椅传动件响度超标 | 优先检查软轴装配而非齿轮修形 | 软轴调整成本低见效快，齿轮修形需要模具改动 |
| 电机 | 有刷电机哒哒声 | 先拆碳刷看磨损 | 80%概率是碳刷问题，拆装5分钟 |
| 齿轮 | 塑料齿轮噪声大 | 检查收缩率补偿是否足够 | 热膨胀导致齿形偏离，非齿轮本身问题 |

### 类型C：决策准则（Decision Rules）

> IF（条件）THEN（推荐行动）BECAUSE（理由）

```
IF 电机噪声且空载出现 THEN 优先查碳刷/电刷 BECAUSE 负载不在时机械噪声通常消失
IF 齿轮箱噪声且速度敏感 THEN 算啮合频率对频率表 BECAUSE 齿轮噪声几乎总是啮合谐波
IF NVH主观评级 <7 THEN 必须采取技术方案 BECAUSE 客户可感知差异
```

### 类型D：踩坑记录（War Stories）

> 项目复盘中的"如果重来一次会怎么做"

| 项目 | 踩了什么坑 | 如果重来 | 教训原则 |
|:-----|:----------|:---------|:---------|
| xxx | 盲目相信仿真结果 | 先做手板样件验证关键假设 | 仿真不能替代实测 |
| xxx | 在齿轮修形上花太多时间 | 先查装配再查修形 | 按概率排序，从低成本到高成本 |
